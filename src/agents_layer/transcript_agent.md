# Meeting Transcription Agent

## Role
One sentence defining exactly what this agent is and what it does.

## Responsibilities
A tight list of what this agent is solely responsible for.
Nothing outside this list is this agent's concern.

## Input
What this agent reads from shared memory and what format to expect.

## Output
What this agent writes to shared memory and the exact format/structure.

## Instructions
Step-by-step rules for how this agent does its job.
Be specific — vague instructions produce vague results.

## Constraints
Hard rules this agent must never break.

## Handoff
What this agent signals when done so the next agent can proceed.
